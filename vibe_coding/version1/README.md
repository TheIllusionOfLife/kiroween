# 90s Website Generator - Version 1

A hilarious vanilla JavaScript generator that creates authentic 90s-style personal websites.

## Features

- 6 retro themes (Neon Dreams, Space Jam, Rainbow Explosion, Matrix Mode, GeoCities Classic, Angelfire Special)
- MIDI background music toggle
- Sparkle cursor trail
- Maximum GIFs mode
- All the classic 90s elements:
  - Blinking text
  - Scrolling marquees
  - Hit counters
  - Guestbook sections
  - WebRings
  - Construction worker emojis
  - Random "Did You Know?" facts
  - Retro badges

## How to Run

Simply open `index.html` in your browser. No build tools required!

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- No dependencies
- Pure 90s vibes
